**** Gioi thieu so qua ve code ***
Trong thu muc BomberMan co 4 file FXML
  MainMenu.fxml : layout cua menu game
  SelectLevel.fxml : layout cua select level
  HighScores.fxml : layout cua high score
  FXMLDocument.fxml : layout chinh - noi game dien ra
Ung voi moi file fxml nay la 4 class Controller tuong ung
Ngoai ra con co class tien ich: Utils, MusicUtils, HighScoreUtils
  Utils: chua cac ham xu ly toa do
  MusicUtils: chua cac ham xu ly am thanh
  HighScoreUtils: chua cac ham xu ly du lieu HighScore (ghi vao file)
Class mainRun.java la noi cua so cua game chay
